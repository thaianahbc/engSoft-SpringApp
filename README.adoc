= Trabalho Final de Arquitetura Web - Eng.Soft

Aplicação construída utilizando React.js e Spring Data REST

Para rodar este aplicativo é preciso ter instalado:
	Spring Tool Suite
	Java 8
	Maven
	
As demais dependências (node) são baixadas de forma automática através do webpack que está configurado no script de inicialização mvmw e pom.xml

Este aplicativo foi feito baseado no tutorial disponiblizado em https://spring.io/guides/tutorials/react-and-spring-data-rest/

O aplicativo contempla a adição, edição e deleção de um Empregado. Os dados não estão sendo persistidos em um banco de dados atualmente.
