= Trabalho Final de Arquitetura Web - Eng.Soft

Componentes do Grupo:
	* Felipe Vilaca
	* Marcus Ricardo
	* Thaiana Hofman

Aplicação construída utilizando React.js e Spring Data REST. Baseado no tutorial disponiblizado em https://spring.io/guides/tutorials/react-and-spring-data-rest/

O aplicativo contempla as ações básicas para manipular um Empregado: Adicionar, Editar e Excluir. 

Dependências para o projeto executar corretamente:
	* Spring Tool Suite
	* Java 8
	* Maven

As demais dependências javascripts são baixadas de forma automática através do webpack que está configurado no script de inicialização mvmw e pom.xml

Após carregar o projeto no STS (Spring Tool Suite) basta rodar como um Spring Boot App, ou pela linha de comando executar o seguinte código: ./mvnw spring-boot:run
